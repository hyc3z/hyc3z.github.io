平行链插槽​

Parachain Slots

波卡作为一个年轻的网络，诞生自Parity Tech自研的Substrate框架。波卡是一个分片(sharding)网络，它的核心理念就是不同的网络各司其职，为了不同的功能而生，而波卡就充当不同网络中间的安全层和通信层。

但是，波卡的资源是有限的，并不是很多人想的那样，像一个k8s的Infra上面可以一直跑各种各样的Instance。区块链为了满足安全的需要，开销是巨大的，这也是去中心化带来的缺点。波卡在19年5月开始拍卖平行链插槽，一开始是5个插槽，预计后续要扩展到50~200个。（后来又修正为100个）

为了保证公平，还用了Candle-Auction, 也就是拍卖的时长是随机的，防止last-second bidding。

这些slot不是永久的，而是通过拍卖得到一个总时长为2年的使用期限。而所谓的拍卖，也并不是拍卖，只是质押一定数量的dot，到期时会自动退还。也就是说，如果从dot本位去看，拍卖本身是免费的。

同时，也没有某种机制来屏蔽、移除平行链。
